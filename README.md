## 抖音直播销售数据分析报告（实习项目示例）

本项目模拟一份 **直播电商数据分析实习作业**，以「巴依老爷」品牌抖音直播为背景，使用 `pandas + matplotlib` 对直播场次进行基础分析和可视化。

### 一、环境准备

- **Python 版本**：推荐 3.8+
- **依赖安装**：

```bash
pip install pandas matplotlib
```

### 二、数据说明（CSV 字段）

默认读取文件：`douyin_live_data.csv`  
首行是表头，示例字段如下（可以在示例基础上手动加几行）：  

- `场次`：第几场直播（整数）
- `成交金额`：每场直播的 GMV（元）
- `累计在线人数`：直播间累计在线人数
- `转化率`：下单人数 / 进入人数，例如 0.215 表示 21.5%
- `ROI`：投流回报率（GMV / 投放成本）
- `GPM`：每千曝光成交金额（元 / 千曝光）
- `曝光次数`：该场直播获得的曝光次数
- `流量来源`：如「自然流量」「品牌自播」「付费投流」等

项目第一次运行时，如果当前目录下没有 `douyin_live_data.csv`，脚本会自动生成一份 **示例数据** 文件，便于直接跑通流程。你可以在这份示例数据的基础上 **手动增加几行** 模拟更多场次。

### 三、脚本功能介绍

主脚本：`douyin_live_analysis.py`

核心流程：

1. **读取 CSV 数据**
2. 使用 `pandas` 计算：
   - 平均转化率
   - 总 GMV（成交金额总和）
   - ROI 趋势（按场次）
3. 使用 `matplotlib` 输出 3 张图：
   - 成交金额趋势折线图：`figures/gmv_trend.png`
   - 转化率柱状图：`figures/conversion_rate_bar.png`
   - 流量来源饼图：`figures/traffic_source_pie.png`
4. 在控制台打印一段 **中文分析结论**，风格参考 211 学生电商运营/数据分析实习报告。

### 四、如何运行

在项目根目录下执行：

```bash
python douyin_live_analysis.py
```

运行结束后，你将看到：

- 终端中输出一段「抖音直播销售数据分析报告（实习版）」文字结论
- `figures/` 目录下生成三张 PNG 图片，用于撰写复盘 PPT 或实习周报

### 五、可以延伸的方向（可写在报告里）

- 加入 **时间维度**（日期、开播时段），分析黄金时段
- 更细的 **投流维度**：不同广告计划 / 人群包 / 创意的 ROI 对比
- 结合 **货品结构**（客单价、类目），看不同组合下的转化率变化
- 尝试导出为 **Markdown / PPT 模板**，自动生成日报/周报

这样整理之后，整体风格更贴近日常实习项目，方便直接放进简历或实习汇报材料中使用。
